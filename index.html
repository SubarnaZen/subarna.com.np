🚀 GitHub Pages + Cloudflare DNS setup
Step 1 — Fix your DNS in Cloudflare
Delete the A record with name a (it points to the wrong subdomain). Add 4 A records with:
Name = @   Proxy = OFF (grey cloud ☁)
185.199.108.153
185.199.109.153
185.199.110.153
185.199.111.153
Add a CNAME: Name = www → Value = YOUR-USERNAME.github.io (proxy OFF)

Step 2 — GitHub Pages
Repo → Settings → Pages → set custom domain = subarna.com.np → tick Enforce HTTPS.

Step 3 — Wait
DNS propagation takes 1–24 hours. Test at dnschecker.org.
⚠ Your current record has name "a" not "@". Also keep Cloudflare proxy OFF for GitHub to verify your domain ownership.
